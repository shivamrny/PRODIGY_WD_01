# PRODIGY_WD_01

## Responsive Landing Page

This project was developed as Task-01 of my Web Development Internship at Prodigy InfoTech.

The objective of this task was to create an interactive and responsive landing page with a fixed navigation menu. The navigation bar remains visible while scrolling, changes its appearance on scroll, and includes hover effects on menu items.

## Features

* Fixed navigation bar
* Hover effects on navigation links
* Navigation style changes on scroll
* Responsive layout for different screen sizes
* Built using HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript
